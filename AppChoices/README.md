# AppChoices Implementation Instructions
The [AppChoices Implementation document](AppChoices%20Implementation.md) provides instructions for DAA participating companies to integrate with AppChoices, the DAA's free mobile application that allows users to exercise choices regarding interest-based advertising that occurs within applications on their mobile device.

AppChoices is the mobile application counterpart to the DAA's WebChoices tool. While WebChoices allows consumers to exercise choices regarding interest-based advertising in web browsers, AppChoices enables similar transparency and control for interest-based advertising within mobile applications.

## Key Concepts

### Purpose
Provides consumers with a destination to:
- Identify participating companies that have committed to responsible advertising practices under the DAA Self-Regulatory Principles.
- Receive a list of companies engaging in interest-based advertising (IBA) within mobile applications on their device using mobile advertising identifiers (e.g., device advertising IDs).
- Exercise choice by opting out of the collection and use of mobile app data associated with mobile advertising identifiers for IBA from one, several, or all listed companies. Future versions of the tool may also support opt-in functionality.

### Tool Access
AppChoices is available through the Google Play Store and the Apple App Store.

### Integration Outreach
Companies interested in being listed in AppChoices should contact Jamie Monaco (jamie@aboutads.info).
